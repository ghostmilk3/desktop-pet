# 🍀 小放光明 · 桌宠工坊（TwinkleFun Desktop Pet）

一个可以把你的"赛博物料"变成桌面小宠的工具 ✨\
支持拖拽图片、自由调整大小、多图片随机切换 🐾

------------------------------------------------------------------------

# 💫 一、这个工具可以做什么？

👉 把你喜欢的图片/GIF变成桌面宠物

你可以：

-   🐻 放小熊
-   🦊 放小狐狸
-   💖 放你的图片创作和约稿

-   请注意，上传的图片最好是透明底图片！

让它们在桌面上陪你一起待着 ✨

------------------------------------------------------------------------

# 🍀 二、如何使用（超级简单）

## ① 启动程序

``` bash
python main.py
```

------------------------------------------------------------------------

## ② 选择图片

-   📁 点击【选择图片】
-   或 🍀 直接拖图片进去

支持：

-   PNG / JPG / GIF（推荐）

------------------------------------------------------------------------

## ③ 调整大小

选中图片 → 拖动滑块

👉 批量统一：

🍀 批量统一大小

------------------------------------------------------------------------

## ④ 点击生成

✨ 召唤桌宠

------------------------------------------------------------------------

## ⏳ 加载说明

加载过程可能短暂无反应，这是正常现象\
看到"正在处理第X张图片"说明正在运行 ✔\
静态图转伪动态图所需时间较久，请耐心等待

------------------------------------------------------------------------

# 🐾 三、操作

  操作   功能
  ------ -----------
  左键   暂停/继续
  拖动   移动
  右键   退出
  空格   退出

------------------------------------------------------------------------

# ✨ 四、功能

-   🍀 多图支持
-   🎚 批量统一大小
-   🐾 自动移动
-   🎞 GIF动画
-   💫 静态图动效

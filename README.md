# 🐾 桌面宠物（Desktop Pet）

一个简单好玩的桌面小宠物程序 🐱  
支持拖动、自动移动、点击暂停等功能。

---

# 🎯 一、如何使用

## 📁 第一步：准备 GIF 图片

1. 在项目目录下找到或创建一个文件夹：

pictures

2. 把你的 GIF 动图放进去

3. 重命名为：

1.gif
2.gif
3.gif
...

⚠️ 注意：
- 必须是 `.gif` 格式
- 名字必须按顺序编号（1、2、3...）

---

## 🧠 第二步：修改代码中的 GIF 列表

打开 `main.py`，找到这一段：

self.gif_paths = [
    os.path.join(base_path, "1.gif")
]

---

如果你有多个 GIF：

self.gif_paths = [
    os.path.join(base_path, "1.gif"),
    os.path.join(base_path, "2.gif"),
    os.path.join(base_path, "3.gif"),
    ......
]

---

# ▶️ 三、运行程序

python main.py

程序启动后需要先加载 GIF 动画资源，运行后一段时间（时间由GIF数量和图片大小决定）会没有反应，没有动画出现
这是正常现象，只要没报错，就请耐心等待，加载完成后桌面宠物会自动出现。
---

# 🕹️ 四、操作说明

左键点击：暂停 / 继续移动  
鼠标拖动：拖动宠物  
右键点击：退出程序  
空格键：退出程序  

---

# ✨ 五、功能说明

- 自动移动
- GIF动画播放
- 随机切换动作
- 支持拖动
- 点击暂停

---

# ⚠️ 六、注意事项

GIF 不显示？
- 检查是否在 pictures 文件夹
- 文件名是否正确
- 是否写入 gif_paths

程序打不开？
- 可能被杀毒软件拦截
